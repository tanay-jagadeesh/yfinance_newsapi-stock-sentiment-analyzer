# News Sentiment Analyzer for Stock Predictions

Collects financial news articles for major stocks, analyzes sentiment using NLP, and correlates with price movements to predict next-day stock direction.

## Tech Stack

**Data & Storage:** News API, yfinance, SQLite, pandas
**NLP & Sentiment:** VADER (vaderSentiment), NLTK
**Machine Learning:** scikit-learn, scipy, NumPy
**Visualization:** Streamlit, Plotly, Matplotlib, seaborn
**Deployment:** Streamlit Cloud

## Features

- Multi-stock news collection and sentiment analysis
- Time series feature engineering (lagged sentiment, price momentum, RSI)
- Correlation analysis with statistical significance testing
- Machine learning models for price prediction
- Interactive dashboard for visualization
